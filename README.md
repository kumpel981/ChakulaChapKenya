# ChakulaChap Kenya

AI-powered meal advisor for Kenyan university students.

## Features
- Find affordable meals on campus
- Budget-based recommendations
- WhatsApp bot integration
- M-Pesa payments
- Admin dashboard
- Firebase database

## Setup

1. **Frontend:**
   - Upload `public/` folder to GitHub Pages
   - Update Firebase config in `app.js`

2. **Backend:**
   - Deploy `server/` to Railway
   - Add environment variables

3. **Firebase:**
   - Create Firebase project
   - Enable Firestore & Authentication
   - Add your config
